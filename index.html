改定ポイント9

計画策定料2（150点）が新設。周術期・回復期とも計画策定料1/2へ区分整理され、病院連携の収益源に。

R8で廃止された施設基準・加算
以下の施設基準が廃止（届出不要に）。該当する医院は運用変更が必要です。
  • 口腔細菌定量検査 ─ 施設基準廃止（届出なしで算定可能に）
  • 咀嚼能力検査 ─ 施設基準廃止
  • 咬合圧検査 ─ 施設基準廃止
  • 光学印象歯科技工士連携加算 ─ 加算自体が廃止
  • 在宅歯科医療推進加算 ─ 廃止（在支診加算1・2に再編）
  • 歯周病重症化予防治療 ─ 廃止（歯周病継続支援治療に統合）
また、以下の定例報告が廃止: 選定療養及び歯科衛生実地指導等の実施状況報告書、情報通信機器を用いた診療に係る報告書、初診料の注1の施設基準に係る報告書（毎年8月の定例報告が不要に）。

制度運用のポイント

対象職員数や算定回数に大きな変動がある場合は、区分の再計算・届出が必要。実績報告で賃金改善額と算定収入額の整合が確認される。

なぜ「2年度平均」という表記か？ ─ R8改定は2年間の診療報酬を一括で改定する設計。R8年度とR9年度で段階的に引き上げることで、医療機関の賃上げが翌年度以降も継続するインセンティブを内蔵している。

経営への含意：改定率の高さは「取りに行く」改定を意味する。新設加算や施設基準の整備を能動的に進めた医院と、何もしなかった医院の収益格差が過去改定以上に広がる構造。

（2）2040年頃を見据えた医療提供体制の構築：かかりつけ歯科医機能の評価、栄養管理・口腔管理による高齢者の生活を支えるケアの推進、在宅医療・訪問看護の確保。

（4）効率化・適正化による制度の安定性向上：後発医薬品使用促進、費用対効果評価制度の活用、施設基準の一部撤廃・定例報告の廃止等。

(Ⅱ)：評価区分の届出で運用する上乗せ枠。R8・R9で点数構造が変わるため、毎年6月時点の区分確認と必要時の再届出を前提に運用する。

実務では「(Ⅰ)の安定運用→(Ⅱ)の区分最適化」の順で整備すると、返戻・届出漏れリスクを抑えやすい。

解説ポイント

届出区分の再計算が必須：対象職員が増えるため、給与総額の基礎数値が変わる。R6で届け出た区分をそのまま使えない可能性が高い。厚労省の計算支援ツールで速やかに再シミュレーションすべき。

賃金の使途の範囲：基本給等の引き上げに加え、それに伴う賞与・時間外手当・法定福利費（事業者負担分を含む）等の増加分に充てることが可能。R8からは夜勤手当の増加額も基本給等に含めて差し支えない。

落とし穴：対象拡大で見かけの賃上げ率が下がる可能性がある。職員数が増えても総支出が変わらなければ、1人あたり賃上げ額が薄まる。ベースアップ収入増と賃上げ支出のバランスシートを毎月確認する体制が不可欠。

経過措置に注意：旧か強診の届出をしていた医院に対する経過措置期間があるが、この間に新要件を満たす体制構築が必要。

歯周治療

SPT/P重防の一本化 ─ 「歯周病継続支援治療」へ

10年以上続いた二重構造が解消。レセコン・運用の全面改修が必要
R6→R8 改定経緯

SPTの歴史：SPT（Supportive Periodontal Therapy：歯周病安定期治療）は2008年（平成20年）に保険導入。歯周基本治療後の安定期に継続管理を行う算定区分として定着した。

P重防の登場：2018年（平成30年）にP重防（歯周病重症化予防治療）が新設。SPTが「治療後」の管理であるのに対し、P重防は「重症化予防」に焦点を当てた区分。ただし現場ではSPTとP重防の使い分けが不明確で、算定ルールの複雑さが問題視されていた。

令和6年度（2024年）：SPT・P重防は併存のまま。ただし中医協の議論で「体系の簡素化」が繰り返し求められていた。

令和8年度（2026年）：歯周基本治療後の包括的治療である歯周病安定期治療及び歯周病重症化予防治療の統合。「歯周病継続支援治療」に一本化。加えて、主治医から紹介された糖尿病患者への歯周病管理に係る加算の運用も見直し。

運用変更のインパクトが最大級：算定コード・名称が全面変更されるため、レセコンマスターの改修規模が今回の改定で最大。カルテテンプレート、算定フロー、スタッフ教育のすべてを刷新する必要がある。

移行期の返戻リスク：6月施行直後に旧コードで請求すると即座に返戻。移行期専用のチェックリストを作成し、全スタッフに周知徹底すること。

P重防の廃止に注意：P重防は廃止される。P重防で管理していた患者は歯周病継続支援治療に移行する。移行時のカルテ記載・管理計画の更新を忘れないこと。

点数面の評価：一本化後の点数は告示で確定。従来のSPT各区分と比較して概ね同水準〜微増だが、P重防からの移行ケースでは算定額が変動する可能性がある。

事務簡素化

事務等の簡素化・効率化 ─ 施設基準廃止と事前承認撤廃

保険適用の事前承認の対象縮小、施設基準の一部撤廃・定例報告の廃止
R6→R8 改定経緯

従来の課題：歯科の保険診療では、多くの処置・材料について地方厚生局への事前承認（理由書・模型・X線フィルム等の提出）が必要とされてきた。これが歯科特有の事務負担として長年問題視されていた。

令和6年度（2024年）：一部の施設基準の簡素化が進んだが、事前承認制度や定例報告の根本的な見直しには至らなかった。

令和8年度（2026年）：保険適用についての事前承認を求めることとされているもののうち、通知等で明確化されているものを事前承認の対象から除外。判断を「事前承認」から「保険適用の可否」に変更。施設基準の一部撤廃や定例報告の廃止も実施。

事前承認の対象から除外される項目

歯冠修復及び欠損補綴の通則で事前承認を求めていた以下の項目等について、通知等で明確化されているものを対象から除外：

  • クラウン・ブリッジ維持管理料の対象で装着した補綴物の再装着（歯周病・外傷・腫瘍等による場合は事前承認不要に）
  • 有床義歯において小児義歯を適用する場合（先天性疾患以外の後継永久歯がない場合）
  • 矯正・先天性欠如等により特定のブリッジを適用する場合

※イからハまで以外の場合は引き続き事前承認が必要。ただし「事前承認」から「保険適用の可否について判断を求める」という表現に変更。

施設基準の廃止・定例報告の廃止

施設基準廃止（届出なしで算定可能に）：

  • 口腔細菌定量検査の施設基準 → 届出不要で算定可能
  • 咀嚼能力検査・咬合圧検査の施設基準 → 届出不要で算定可能
  • 在宅歯科医療推進加算の施設基準 → 廃止（加算自体を廃止）

定例報告の廃止：

  • 歯科点数表に係る解釈が示されていない項目の明確化
  • 内容が類似する項目や複数年にわたり算定実績がない項目の整理
  • 算定告示と算定要件が一致していない項目の整理
解説ポイント

実務上のメリット：事前承認の対象縮小により、理由書の作成・模型の提出等の事務作業が軽減。特にブリッジ・義歯関連の事前承認が不要になるケースが増えるため、患者への提供スピードが向上する。

施設基準廃止の影響：口腔細菌定量検査・咀嚼能力検査・咬合圧検査の施設基準が廃止されることで、これまで届出をしていなかった医院も即座に算定可能に。検査機器の導入判断が純粋に経営的観点からの判断になる。

注意点：「施設基準の廃止」は「点数の廃止」ではない。あくまで届出不要で算定可能になるだけで、点数自体は存続する。一方、光学印象歯科技工士連携加算や在宅歯科医療推進加算は加算そのものが廃止される。

新製有床義歯管理料の算定単位見直し、クラスプ材料のコバルトクロム原則化
R6→R8 改定経緯

金パラ問題の背景：金銀パラジウム合金（金パラ）は長年にわたり歯科保険診療の主要材料だったが、近年の価格高騰で逆ザヤリスクが課題となった。

代替材料の推進：2020年以降、CAD/CAM関連材料やコバルトクロム合金への切り替えが段階的に推進されている。R4改定ではCAD/CAM冠・インレー等の対象拡大が進んだ。

令和6年度（2024年）：金パラは依然として選択肢の一つ。クラスプについても複数材料が並列で設定されている。

令和8年度（2026年）：部分床義歯のクラスプ・バーの材料について、歯科用貴金属材料の価格状況への対応として見直し。有床義歯管理の評価体系についても算定単位や評価体系を見直し。

解説ポイント

コバルトクロム原則化の実務：コバルトクロムが標準材料となり、金パラは「例外」の位置付けに。委託先技工所がコバルトクロムでの製作体制を整えているか確認が必須。材料費は大幅に低減するが、加工の難易度が異なるため技工料への影響も考慮。

クラウン・ブリッジ維持管理料の対象整理：維持管理料の対象となる補綴物の範囲が整理された。CAD/CAM冠の適用拡大に伴う対応。

3Dプリント有床義歯：デジタル技術を活用した新たな有床義歯の製作法が保険適用に。設備投資の判断と技工所の対応状況の確認が必要。

答申書附帯意見：今後、歯科技工料調査の結果等を踏まえた歯冠修復及び欠損補綴等の評価の見直し、臨床現場の実態等を踏まえた評価の見直しが引き続き検討される。

11 ─ 医療DX

医療DX関連加算の再編

旧加算の整理と新加算への移行で、届出・運用の再確認が必要
R6→R8 改定経緯

令和6年度（2024年）：歯科では「医療DX推進体制整備加算（初診11/10/8点、再診9/8/6点）」で段階評価され、オンライン資格確認・電子処方箋対応等が求められた。

令和8年度（2026年）：名称と体系を整理し、電子的歯科診療情報連携体制整備加算へ再編。初診時の評価区分と再診時の評価が見直された。

運用上の論点：マイナ保険証利用率要件、電子処方箋の導入状況、施設基準の届出内容が算定可否を左右する。

解説ポイント

移行対応：R6で旧DX加算を算定していた医院は、新加算への切替要件を再点検し、届出区分を見直す必要がある。

実務対応：オンライン資格確認・マイナ保険証運用の院内フローを標準化し、受付〜会計までの運用手順を統一すると算定漏れと返戻を防ぎやすい。

優先順位：まずは現行要件で算定可能な区分を確実に取得し、電子処方箋等の追加要件は段階導入で対応するのが現実的。
