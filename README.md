# r8-dental-report-chatbot-lab

R8歯科改定レポートの**テスト用**チャットボット埋め込み版です。  
本番サイトとは分離して検証する用途です。

## できること
- ページ右下の `AIに質問` からチャットを開く
- サイト内テキストを優先して関連セクションを抽出
- Gemini APIキーを入力すると回答生成
- APIキー未入力時は「関連セクション案内モード」で動作

## 使い方（無料枠前提）
1. Google AI Studio で Gemini API Key を取得
2. ページ内チャットのAPIキー欄へ入力
3. 質問を入力して送信

## 注意
- このテスト版はAPIキーをブラウザの`localStorage`に保存します（本番用途には非推奨）
- 生成回答はサイト内情報のみを優先するプロンプトにしていますが、最終確認は原資料で行ってください
